# Introduction

Carrier Rate Calculation and Shipping Label Generation API

<aside>
    <strong>Base URL</strong>: <code>http://api-project.test</code>

    Welcome to the **Logistics API**.

    This POC demonstrates a high-performance shipping engine.
    - **Smart Carrier Selection:** Finds the cheapest rate with weight tie-breaking.
    - **Background Processing:** Labels are generated via Laravel Queues.
    - **Reliable Validation:** Strict weight limits and error handling.
